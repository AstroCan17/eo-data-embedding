# Configuration item data list

## Introduction

This section constitutes the Configuration item data list (CIDL) for the
eo-data-embedding project.

`eo-data-embedding` is a single-package Python library with exactly one
configuration item: the `eo_data_embedding` package, versioned through its
`pyproject.toml` / `__version__` and the corresponding git tags. A formal,
fully indentured CIDL (part-number breakdowns, drawing issue status, PMP lists)
is therefore tailored out and is not part of the SDP §3 document tree; this page
records only the online documentation baseline that constitutes the delivered
configuration item.

## Applicable and reference documents

The applicable and reference documents supporting this configuration item are
the artifacts of the online documentation, listed under
[Online documentation](#online-documentation) below.

### Reference documents

## List

The configuration item is the `eo_data_embedding` Python package and its
accompanying online documentation. As a single-package software-only library it
has no indentured part-number breakdown, no customer-controlled hardware
specifications and no PMP lists, so those CIDL entries are tailored out. The
configuration item is identified by the online documentation set listed under
[Online documentation](#online-documentation), each entry being defined by its
title and a link to the released document.
